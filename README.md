# Neon Circuit Portfolio

React + Vite static portfolio built for GitHub Pages.

## Run locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Deploy options

- Automatic: Push to `main` and use `.github/workflows/deploy.yml` (GitHub Actions Pages).
- Manual: `npm run deploy` to publish `dist` to `gh-pages` branch.

## Content updates

- Main page content: `src/App.jsx`
- Styling and theme: `src/App.css`
- Resume file: `public/Jashwanth_Bamidi_Resume.pdf`
